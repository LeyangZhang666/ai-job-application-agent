# Autonomous Career Agent

An AI-agent powered autonomous job application system designed to automate the end-to-end recruiting workflow across LinkedIn, Greenhouse, Lever, and company career portals.

## Core Features

- AI-powered job discovery
- Resume tailoring based on JD parsing
- Automated browser execution
- Email verification code retrieval
- Auto-registration for external platforms
- Multi-agent task coordination
- Application tracking system

## Workflow

Job Discovery Agent
↓
Resume Optimization Agent
↓
Browser Automation Agent
↓
Email Verification Agent
↓
Application Tracking Agent

## Tech Stack

- OpenAI GPT models
- Browser automation
- Multi-agent workflow orchestration
- Long-chain reasoning
- Email parsing automation

## Current Progress

The system is currently being tested for high-frequency cross-platform job applications and significantly reduces manual application time.
